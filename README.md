# DevTools - Free Online Developer Tools

A collection of 20+ free online developer tools for everyday use.

## Tools

### Encode / Decode
- Base64 Encode/Decode
- URL Encode/Decode
- HTML Entity Encode/Decode
- JWT Decode
- Hash Generator (MD5, SHA-1, SHA-256)

### Formatter
- JSON Formatter
- JSON Minify
- SQL Formatter
- CSS Minifier
- Regex Tester

### Generator
- UUID Generator
- Password Generator
- QR Code Generator

### Converter
- Color Converter (HEX ↔ RGB)
- Number Base Converter (Binary, Octal, Decimal, Hex)
- Timestamp Converter
- Case Converter

### Utilities
- Word Counter
- Diff Checker
- IP Lookup

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Deployment**: Cloudflare Workers
- **Analytics**: Google Analytics 4

## Development

```bash
npm install
npm run dev
npm run build
```

## Deployment

Deploy to Cloudflare Workers using `@cloudflare/next-on-platform`:

```bash
npm run deploy
```
